auth.shopify-session-token

Shopify Session Token Verification

Middleware that verifies the short-lived JWT issued by Shopify App Bridge for every request from the embedded app. Validates signature, claims, and attaches shop context to the request.

Use it / Avoid it

Use when
  • Building an embedded Shopify app inside the Admin iframe.
  • Any API endpoint called by the embedded frontend.
  • Mentions “session token”, “app bridge”, “jwt verification”.
Avoid when
  • Non-embedded storefronts — use integration.shopify-app-proxy.
  • Webhook handlers — HMAC body signing, different flow.
  • Server-to-server calls when you already hold an offline access token.
