Shopify App Installation & OAuth

The OAuth handshake every Shopify app must implement. Merchant approves install, app exchanges authorization code for an offline access token, and the token is persisted for Admin API access.

Overview

The OAuth handshake every Shopify app must implement. Merchant approves install, app exchanges authorization code for an offline access token, and the token is persisted for Admin API access.

Prerequisites

This block has no prerequisites — it can be implemented standalone.

Data Model

Logical tables introduced by this block. Types resolve to dialect-specific SQL per the canonical logical-type mapping.

shops
Merchant record with encrypted access token, granted scopes, install/uninstall timestamps.
oauth_nonces
Single-use CSRF tokens for the OAuth callback (5 min TTL).

Use it / Avoid it

Use when
  • App needs to be installed on a Shopify store.
  • Mentions “shopify app”, “install app”, “oauth”, “access token”.
  • App needs to call the Shopify Admin API on behalf of a merchant.
Avoid when
  • Building a Shopify theme — no OAuth needed.
  • Building a sales channel — different auth flow.
  • Need per-request auth for an embedded app — use auth.shopify-session-token.
