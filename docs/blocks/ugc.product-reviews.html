Product Reviews

Shopper-submitted product reviews with rating, optional title and body, verified-buyer attribution, moderation workflow, auto-approve threshold, and a denormalized aggregate for fast PDP rendering.

reviewsugcsocial-proofmoderationratings

Overview

Shopper-submitted product reviews with rating, optional title and body, verified-buyer attribution, moderation workflow, auto-approve threshold, and a denormalized aggregate for fast PDP rendering.

Prerequisites

This block has no prerequisites — it can be implemented standalone.

Data Model

Logical tables introduced by this block. Types resolve to dialect-specific SQL per the canonical logical-type mapping.

reviews
Per-customer-per-product review with status state machine, server-computed verified_buyer, moderation metadata.
review_aggregates
Denormalized rollup updated transactionally on moderation actions. Cheap reads, eventual staleness avoided.

Use it / Avoid it

Use when
  • Mentions “reviews”, “ratings”, “social proof”.
  • Product pages need customer feedback enrichment.
  • Merchant wants moderation control over UGC.
Avoid when
  • Q&A on products — use a different UGC block.
  • Site-wide testimonials — not product-scoped.
  • Photo or video-only review — extend this block with media.
